## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/cdrcknt/philippines-calendar-2025.git
cd philippines-calendar-2025
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```
